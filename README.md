# 🔐 Auto Pentest Tool

An automated penetration testing tool designed to scan web applications, identify vulnerabilities, and generate structured reports.

Built for security researchers, students, and professionals.

---

## 🚀 Features

- 🔍 Automated vulnerability scanning  
- 📊 PDF report generation  
- ⚡ Live scan progress tracking  
- 🌐 Web application testing  
- 🧪 SOAP/WSDL attack module  
- 📝 Logging and result storage  

---

## ⚙️ Installation

```bash
git clone https://github.com/DharmendraSoni5110/Crowintel.git
cd Crowintel
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
chmod +x run.sh
▶️ Usage
./run.sh
